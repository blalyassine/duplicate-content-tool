A simple plugin that adds an option to quickly duplicate posts and pages in WordPress.

== Description ==
Duplicate Content Tool allows users to quickly create copies of their posts and pages. This plugin is ideal for content creators, bloggers, and site administrators who want to save time and effort when replicating content.

== Installation ==
1. Upload the plugin folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.

== Frequently Asked Questions ==
= Does this plugin work with custom post types? =
No, currently, the plugin only supports default posts and pages.
